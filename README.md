# Maya Assistant
###### IA made for discord

#### Configuration
Create an **.env** file with : 
- DISCORD_TOKEN=<your_token>
- API_URL=<your_url>
- KEY=<your_key>
- BID=<code_BID>
- HOST=<host_name>
- API_KEY=<your_api_key>

path of file : Maya/.env

#### Logs
Create a folder called logs in **Maya/logs**

#### Virtual Environment
Add a virtual environment (venv) to the project
